  4. chaque enseignement noté sur 5

Matériel

  1. L'énoncé est un fichier html sujet-a-lire dans titan:pub/progweb-serveur/tpjee-groupeA
  2. Serveur apache Tomcat ou Glassfish
  3. zip contenant quatre fichiers ; titan:pub/progweb-serveur/tpjee-groupeA/
    1. tpjees4.sql script sql pour installer la base en local
    2. bd.jsp script pour se connecter à la bd et afficher les résultats
    3. index.jsp la page de login
    4. vote1.jsp la page de vote pour les maths
  4. sur le serveur 192.168.2.192, la base est déjà installée, pour se connecter avec le couple (tpjee,tpjee)

A rendre

Vous devez remettre un zip complet de votre application sur e campus 2 : espace Prog Web

Scénarii

  1. (user,user) se connecte pour voter
    1. le vote se fait sur trois pages (trois matières)
    2. possibilité de revenir en arrière (lien hypertexte)
    3. la validation à la dernière page provoque l'insertion des données en BD et la déconnexion (retour à la page de connexion)
  2. (admin,admin) se connecte pour voir le bilan des votes et la moyenne par matière (à calculer), il se déconnecte
